# SpecEcho Diff v0.1.0

SpecEcho Diff v0.1.0 is the first public MVP for runtime OpenAPI contract comparison.

## Highlights

- Parse OpenAPI 3.x YAML/JSON and resolve local schema references.
- Probe explicitly allowed HTTP targets and compare real JSON responses.
- Detect breaking status-code, missing-field, type, and enum changes.
- Compare staging and production for semantic drift.
- Generate Markdown, JSON, and standalone HTML reports.
- Ship deterministic FastAPI fixture services and an offline demo report.

## Assets

- Python wheel
- Source distribution
- Example report bundle
- SHA256 checksum manifest

## Validation

The release checklist requires lint, typecheck, tests with coverage, package build, deterministic demo generation, release packaging, and contributor identity verification before publication.
